# Pawfect Pals 🐾

Pawfect Pals is a modern pet care and shopping platform built using HTML, CSS, and JavaScript. The project provides a user-friendly and interactive interface where pet lovers can explore products, manage shopping carts, browse categories, and access company support information through a clean and responsive design.

The platform is designed to deliver a smooth e-commerce experience for pet owners while focusing on simplicity, accessibility, and modern UI aesthetics.



## ✨ Features

- 🛍️ Product category browsing
- 🔍 Real-time product search
- 🐶 Multi-category pet product support
- 🛒 Dynamic shopping cart system
- 💳 Checkout summary interface
- ➕ Quantity management in cart
- ❌ Remove products from cart
- 📦 Product detail modal popup
- 👤 User-specific cart using LocalStorage
- 📱 Responsive and modern UI design
- 📄 Company, Legal & Help information sections



## 🛠️ Technologies Used

| Technology | Purpose                    |
|------------|----------------------------|
| HTML5 | Structure                       |
| CSS3 | Styling & Layout                 |
| JavaScript | Dynamic Functionality      |
| LocalStorage | Cart & User Data Storage |
